## 手写 new 的执行过程
首先我们知道 new 的执行过程如
1. 创建一个空对象
2. 将对象的 ```__proto__``` 指向构造函数的 prototype
3. 将这个对象作为构造函数的 this
4. 确保返回值为对象

```js
function myNew(Con,...arg) {
  let obj=Object.creat(Con.prototype)
  let result=Con.apply(obj,arg)
  return result instanceof Object?result:obj

}

```
