## vue 项目长列表
长列表一般也叫虚拟列表，是一种大数据量下只渲染可见节点避免页面卡顿的优化方案

了解什么是长列表，可以看这篇文章[「前端进阶」高性能渲染十万条数据(虚拟列表)](https://juejin.im/post/5db684ddf265da4d495c40e5)
